A.User, Post, and Comment tables.

1.CREATE TABLE User (
    user_id INTEGER PRIMARY KEY,
    username VARCHAR(255),
    email VARCHAR(255),
    password VARCHAR(255)
);

2.CREATE TABLE Post (
    post_id INTEGER PRIMARY KEY,
    title VARCHAR(255),
    content TEXT,
    created_at DATETIME,
    user_id INTEGER
);

3.CREATE TABLE Comment (
    comment_id INTEGER PRIMARY KEY,
    text VARCHAR(255),
    created_at DATETIME,
    user_id INTEGER,
    post_id INTEGER
);


B.Query to find all posts created by a specific user,
including the user details.


SELECT *
FROM Post
INNER JOIN User ON Post.user_id = User.user_id
WHERE User.user_id = [specific_user_id];
